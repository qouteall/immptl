    # How to Install This Mod (Fabric version)

    1. Install Fabric Loaderopen in new window
    2. Open the game directory, find the mods folder, put the mod jar into mods folder. (If mods folder does not exist, launch the game with Fabric once and the folder will be created.)
    3. Download Fabric API (from GitHubopen in new window or CurseForgeopen in new window or Modrinthopen in new window), put Fabric API mod jar into mods folder. (Fabric API is a mod. Fabric API is not Fabric loader)
